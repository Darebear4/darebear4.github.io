This website was first created for the IT&C 210A lab 1 assignment. In the first part, lab 1A, I was tasked with creating a simple webpage with a To-Do list. This page can be viewed by following the link that says "Old Version" in the navigation bar(Note: the old version has no javascript and does not function. I put that there purely for the satiation of your own curiosity). In the second part, lab 1B, I was tasked with creating a more advanced and aesthetically pleasing version of the same webpage. The newer version can be found by following the link that says "Current Model" in the navigation bar.

In lab 2A and 2B, I coded the javascript that allows this website to store tasks in the browser's local storage. The best part about using local storage is that once you have loaded the site, you can use it without internet access!

Another bonus to local storage is that you don't even have to log in! The tasks are stored in your browser, so you can even close the site and come back to the same tasks later (assuming it's the same browser and it's storage hasn't been cleared).

Note: if you activate the "Filter completed tasks" option, your tasks will not be deleted. You can simply click the toggle to show them again.
